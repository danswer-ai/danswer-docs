## How it works
The Linear connector pulls in all the Linear issues and the associated comments.

Issues are updated every **10** minutes.


## Setting up
### Authorization
    1. Grab an API key as shown below.
![LinearApiKey](/images/connectors/linear/LinearApiKey.gif)

### Indexing
    1. Navigate to the Admin Dashboard and select the **Linear** Connector
    2. In **Step 1**, provide the **API Key** from above.
    3. Click `Connect` to begin indexing
