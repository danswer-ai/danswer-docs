At Onyx, we strive to make our core open-source offering as powerful as possible. However, we understand that large enterprises require additional support and functionality, so we offer additional enterprise-oriented features and functionality.

## EE Features

Here are some of our enterprise-only offerings.

### Security and Access Control

1. **Single Sign-On (SSO)**
   Streamline user authentication and enhance security with SSO integration (OIDC + SAML)

2. **Role-Based Access Control (RBAC)**
   Manage user permissions efficiently with granular control over access levels.

3. **Credential Encryption**
   Ensure the highest level of security for sensitive information with robust encryption.

4. **Document Permission Inheritance**
   Simplify permission management with hierarchical inheritance for documents and folders.

5. **API Key Authentication**
   Secure your API integrations with dedicated API key authentication.

### Enhanced Functionality

6. **Additional APIs**
   Access extended API endpoints for more advanced integrations and workflows.

7. **White Labeling**
   Customize the interface to match your brand identity for a seamless user experience.

8. **Admin Dashboard / Analytics**
   Gain insights into usage patterns and system performance with comprehensive analytics.

9. **Query History**
   Track and analyze user queries for improved auditing and optimization.

10. **Usage Limits Configuration**
    Set and manage usage limits to align with your organization's needs and policies.

### Support and Services

11. **Enterprise-Level Support**
    Benefit from priority support with dedicated account managers and faster response times.

## Roadmap

We're constantly working to expand our enterprise offerings. Here's a preview of some exciting features on our roadmap:

- **Expert Suggestion**
  Leverage AI-powered recommendations to connect users with internal subject matter experts.

- **Personalized Search**
  Deliver tailored search results based on user roles, permissions, and past interactions.

- **Org-Structure Importing**
  Seamlessly integrate your organization's hierarchy for more intuitive permission management and content organization.
